# What is this?
A browser extension that reskins the FlavorTown website with a GitHub theme.

# Is it finished?
No, it is far from complete.

# How to use it
1. Download all files.
2. Open your browser's extension settings and enable Developer Mode.
3. Click "Load unpacked."
4. Select the project folder containing the manifest file, then any file in it.
5. Refresh the FlavorTown page.
